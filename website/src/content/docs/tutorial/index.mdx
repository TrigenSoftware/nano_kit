This tutorial teaches Nano Kit by building a small React application in stages. The demo project is Event Board, a public listing of frontend events.

The project starts as the smallest useful version, then adds Nano Kit packages when the application needs them.

You will move through the same path a real app tends to take:

1. [Build the first store](/tutorial/first-store).
2. [Load data with query](/tutorial/load-data-with-query).
3. [Routing](/tutorial/routing).
4. [Route parameters](/tutorial/route-parameters).
5. [Forms and mutations](/tutorial/forms-and-mutations).
6. [Optimistic updates](/tutorial/optimistic-updates).
7. [Prepare for SSR](/tutorial/prepare-for-ssr).
8. [Server rendering](/tutorial/server-rendering).
9. [Final result](/tutorial/final-result).

The tutorial begins with [Setup](/tutorial/setup).
