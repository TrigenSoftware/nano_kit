This example shows a small public event board built with `@nano_kit/store`, `@nano_kit/query`, `@nano_kit/router`, `@nano_kit/intl`, and Nano Kit SSR integrations.

It covers routed pages, URL filters, an in-memory HTTP API, session auth with SSR cookie forwarding, infinite queries, mutations, optimistic updates, entities, page-level `Stores$`, dynamic `Head$`, and a custom Hono production server.

Demo users: `ada` / `lovelace`, `grace` / `hopper`.

The Nano Kit tutorial uses this app as its running project.
